unset_key

unset_key( $key, $array, $reindex = false )

Description

Unset an array element by key and optionally reindex

Parameters

  • $key [string] - Key to check
  • $array [array] - Array to check
  • $reindex [bool] - Optionally reindex array after removing key

Returns

  • $narr [array] - Array without specified key

Source code

function unset_key( $key, $array, $reindex = false ){

    if( $key !== false) {
        unset( $array[$key] );
        
        if( $reindex == true ){
            $narr = array_values($array); // 'reindex' array
            return $narr;
        }
        
    }
    
}
