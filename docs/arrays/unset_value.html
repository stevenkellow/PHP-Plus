unset_value

unset_value( $value, $array, $reindex = false )

Description

Unset an array element by value and optionally reindex

Parameters

  • $value [string] - Value to check
  • $array [array] - Array to check
  • $reindex [bool] - Optionally reindex array after removing value

Returns

  • [array] - Array without specified value

Source code

function unset_key( $key, $array, $reindex = false ){

    if( $key !== false) {
        unset( $array[$key] );
        
        if( $reindex == true ){
            $narr = array_values($array); // 'reindex' array
            return $narr;
        }
        
    }
    
}
