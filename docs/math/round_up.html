round_up

round_up($number, $precision = 2)

Description

Round up a number to the next whole integer

Note: negative numbers round towards zero

Parameters

  • $number [float] - Float to round up
  • $precision [int] - number of decimal places to round to (default: 0)

Returns

  • [int] - A whole number

Source code

Author: taking sides

function round_up($number, $precision = 2){
    $fig = (int) str_pad('1', ( $precision + 1 ), '0');
    return (ceil($number * $fig) / $fig);
}
