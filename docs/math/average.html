average

average( $array OR $list, $of, $numbers )

Description

Get the average of an array or a list of numbers

Parameters

  • $array [array] - Array of numbers to convert
  • list of numbers [float | int] - a comma separeted list of numbers to average

Returns

  • [float | int] - A number representing the average of all passed arguments

Source code

Author: Mucello [Code heavily modified]

function average(){
  
    // Get all the arguments
    $arg_list = func_get_args();
    
    // If an array was passed through
    if( is_array( $arg_list[0]) ){
        
        return array_sum( $arg_list[0] ) / count( $arg_list[0] );
        
    } else {
        
        // Get number of arguments passed through
        $numargs = func_num_args();
    
        // Create an array for each passed element
        $args_array = array();
        for ($i = 0; $i < $numargs; $i++) {
            $args_array[] = $arg_list[$i];
        }

        return array_sum( $args_array ) / $numargs;
    
    }
    
}
