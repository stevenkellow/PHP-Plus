is_even

is_even( $number )

Description

Check if a number is even

Parameters

  • $number [int] - Number to check

Returns

  • [bool] - True ( number is even ) | False ( number is odd )

Source code

Author: Pawel Dubiel

function is_even( $number ){
    
    if( $number & 1 ){
        return false;
    } else {
        return true;
    }
    
}
